1. Requisitos previos:
   - Tener instalado Python 3.x
   - Instalar Selenium con el comando:
     pip install selenium
   - Tener Google Chrome instalado.
   - Descargar ChromeDriver compatible con la versión de Chrome y colocarlo en el PATH del sistema.

2. Archivos incluidos:
   - viviana_pedraza_e2e_ver1.py (script principal)
   - Capturas de pantalla
       01_producto1_agregado.png
       02_producto2_agregado.png
       03_carrito.png
       04_confirmacion_compra.png
       Archivo  Conclusiones 
       Archivo Readme
   

3. Cómo ejecutar:
   - Abrir una terminal en la carpeta donde está el script.
   - Ejecutar:
     python viviana_pedraza_e2e_ver1.py

4. ¿Qué hace el script?
   - Abre el sitio https://www.demoblaze.com/
   - Agrega dos productos al carrito.
   - Abre el formulario de compra, llena los datos y finaliza la compra.
   - Toma capturas de pantalla
